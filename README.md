# angular-eslint-injection-context

ESLint rules for Angular injection context.

## Requirements

- Angular and Angular ESLint v21
- TypeScript v5.9
- ESLint v10
- TypeScript ESLint v8
- New flat ESLint configuration (`eslint.config.js` or equivalent)

> [!NOTE]
> `.eslintrc.json` and other legacy ESLint configurations are not supported

> [!NOTE]
> It probably works with older versions, but it is not tested and requires to [override dependencies](https://docs.npmjs.com/cli/configuring-npm/package-json#overrides)
